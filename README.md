# OnyXLuxe Tickets

A simple ticket booking website for various events in Nigeria.

## Features

- Homepage with event listings
- User signup and login (demo)
- Contact form
- Ticket purchase page with form validation
- Responsive design

## Pages

- `index.html`: Homepage with event previews
- `Signup.html`: User registration
- `login.html`: User login
- `contact.html`: Contact form
- `purchase.html`: Ticket purchase form

## Usage

1. Open `index.html` in a web browser.
2. Browse events and click "Preview Details" for info.
3. Click "Buy Ticket" to go to the purchase page.
4. Fill out the purchase form and submit.

## Technologies

- HTML5
- CSS3
- JavaScript (vanilla)

## Notes

This is a frontend-only demo. In a production app, you'd need:

- Backend server (e.g., Node.js/Express)
- Database (e.g., MongoDB)
- Payment integration (e.g., Paystack)
- User authentication system
